LOGO FILES FOR PDF LETTERHEAD
================================

Please add your college logo images to this folder:

1. college-logo.png - Left side college logo (recommended size: 200x200 pixels)
2. naac-logo.png - Right side NAAC/Institution logo (recommended size: 200x200 pixels)

These logos will appear on the Internship Approval Letter PDF that teachers generate for students.

Image Format: PNG or JPG
Recommended Size: 200x200 pixels (will be displayed at 70x70 in PDF)
Background: Transparent PNG preferred

Note: If logos are not added, the PDF will still generate but will show broken image placeholders.
